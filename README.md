# Snake Game 🐍

A snake game built using:

- Python Flask
- HTML
- CSS
- JavaScript
- SQLite

## Features

- Snake movement
- Food system
- Score tracking
- Database integration
- Leaderboard

## Run Project

Install Flask:

```bash
pip install flask
```

Run:

```bash
python app.py
```

Open browser:

http://127.0.0.1:5000
